<h1 align="center">Gerenciador de Operações - Casa de Câmbio Muito Dinheiroo</h1>

<h2>Descrição</h2>
<p>Este repositório contém o gerenciador de operações da casa de câmbio Muito Dinheiro, que tem como objetivo facilitar a realização de atividades rotineiras e documentar operações.</p>
<p>Este gerenciador foi escrito em linguagem Python utilizando como IDE o Notebook Google Colaboratory, é compostos em seu primeiro bloco de instruções por uma calculadora de conversões que realiza as operações matemáticas para converter as moedas de acordo com a solicitação do usuário, bem como, fornecer um panorama préveio das operações realizadas. No segundo bloco é possível realizar filtros por nome do cliente, intervalo de data da operação e por nome do cliente e intervalo de data da operação, exibindo ao usuário um banco de dados filtrado e com valores totais das operaçõs realizadas e dsa taxas cobradas. </p>
